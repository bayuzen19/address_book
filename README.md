## Introduction
Zentact is a sleek, modern address‐book CRM designed to help you manage contacts effortlessly. Built with semantic HTML5 and clean, modular CSS, Zentact offers a lightweight, responsive interface for searching, labeling, and organizing your connections in one place.

## Features
- **Instant Search**  
  Quickly find any contact using a live search widget that filters results in real time.  

- **Organize with Labels**  
  Categorize contacts by family, friends, customers, or any custom label to keep your list structured.  

- **Add & Edit Contacts**  
  Use an intuitive form to create new entries or modify existing profiles without refreshing the page.  

- **Secure Deletion**  
  Remove outdated or unwanted entries with a single click, ensuring your contact list stays up to date.

## Project Structure
- **landingPage.html**  
  Main entry point containing the header, hero section, features grid, and footer.  

- **style.css**  
  Central stylesheet that defines global resets, typography, layout rules, and interactive state transitions.  

- **logo.png**  
  Branded graphic displayed in the header and hero sections to reinforce identity.

## Getting Started
1. Clone or download the repository.  
2. Ensure all files (HTML, CSS, logo) reside in the same directory.  
3. Open `landingPage.html` in any modern browser to view the interface.  
4. Navigate through sections or click “Enter Charm” to proceed (placeholder link).

## Customization
- **Branding**  
  Replace `logo.png` with your own graphic to align with your company identity.  

- **Color Palette**  
  Adjust the primary and background colors in `style.css` to match your branding guidelines.  

- **Typography**  
  Swap out the font-family declaration with your preferred web font for a distinct look.

## Responsive Design & Styling
- Utilizes CSS Flexbox and Grid for a fluid layout that adapts to screen width.  
- Mobile-first breakpoints ensure legibility on small devices, while auto-fit grids accommodate wider viewports.  
- Subtle hover and transition effects provide visual feedback without compromising performance.

## Future Enhancements
- **Dark Mode Toggle**  
  Offer a switch to invert colors for low-light environments.  

- **Contact Import/Export**  
  Enable CSV or vCard support to bulk-upload or download address-book data.  

- **User Authentication**  
  Integrate a login system for personalized lists and permission control.  

- **Backend Integration**  
  Connect with a database or API to persist contacts across sessions and devices.
